Detailed Description

Ejercicio 3 de la Práctica.

En este ejercicio se nos pide implementar el algoritmo de sincronización de procesos "Productor-Consumidor" Un proceso es el encargado de producir cierto recurso mientras el otro proceso lo consume, todo esto de forma correctamente sincronizada para aseguranrnos de que se respeta la exclusión mutua en las zonas cŕiticas y asegurandones tambien de que no se produce ningun tipo de interbloqueo.
